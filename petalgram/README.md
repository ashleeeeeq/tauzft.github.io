# Petalgram - Flower Shop Ordering System

A full-stack flower shop ordering system built with PHP and CodeIgniter 4.

## Features
- Product catalog with categories
- Shopping cart and checkout
- Order placement and tracking
- User authentication and profiles
- Admin dashboard

## Installation
1. Clone the repository
2. Run `composer install`
3. Configure `.env` file
4. Import `database/schema.sql` into MySQL
5. Import `database/seed.sql` for sample data
6. Run `php spark serve`
